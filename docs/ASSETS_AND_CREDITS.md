# 素材与鸣谢

Animal Island 是源码公开的自定义 Hexo 主题，仅限个人学习与非商业用途，严禁商业使用。项目视觉采用温暖小岛 UI 风格；本项目不是 Nintendo 官方产品，未获得 Nintendo 背书，也与 Nintendo 无任何关联。

## 主要 UI 来源

### animal-island-ui

- 地址：https://github.com/guokaigdg/animal-island-ui
- 作者：guokaigdg
- 上游信息：其 `package.json` 声明 `license: MIT`，并在描述中标注 `for learning purpose only`。
- 本项目用途：UI 元素来源、视觉风格参考、组件 token 参考，以及部分光标、海浪、树等小岛风格素材来源/改编。
- 说明：这些 UI 元素来自 `animal-island-ui` 及已列明来源，不是 Nintendo 提供的素材。

### ac-site-template

- 地址：https://github.com/yunxinz/ac-site-template
- 作者：Zhang Yunxin / yunxinz
- 上游许可：MIT License
- 本项目用途：布局氛围、横幅/装饰素材参考，部分素材复制或改编并保留鸣谢。

## 本主题素材目录

```text
source/images/island-assets/
```
