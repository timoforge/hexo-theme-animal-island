Animal Island is a source-available custom Hexo theme for personal learning and non-commercial use only. Commercial use is prohibited. The visual style follows a warm island UI mood. This project is not an official Nintendo product, is not endorsed by Nintendo, and is not affiliated with Nintendo.

## 🌿 Main UI sources

### animal-island-ui

- URL: https://github.com/guokaigdg/animal-island-ui
- Author: guokaigdg
- Upstream info: its `package.json` declares `license: MIT` and notes `for learning purpose only`.
- Usage in this project: UI elements, visual references, component token references, and selected island-style assets such as cursor, sea, and tree elements.
- Note: these UI elements come from `animal-island-ui` and listed sources; they are not provided by Nintendo.

### ac-site-template

- URL: https://github.com/yunxinz/ac-site-template
- Author: Zhang Yunxin / yunxinz
- Upstream license: MIT License
- Usage in this project: layout atmosphere, banner/decorative references, and selected copied or adapted assets with credit retained.
